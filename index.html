HTML

  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • check documentation for different things
  • box shadow is a new thing

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Repo = a folder for your project that contains all of the files and their history
  • Branch = create new changes that don’t affect live state. Create a new branch for your work
  • Markdown files = the way content files are formatted in github. A way to write for the web using plain text, converts to html
  • Commit = saving a new version of file you’re working on
  • Pull request = asking for changes on a branch to merge into another branch. Way to discuss and review your work with others.
  • Merge = when a pull request is approved the changes are merged into the main branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • to declare a for loop : for(var i = 0; i < j; i++)
  • to declare a function: function functionName() { "code goes here "}
  • to call a function: functionName();
